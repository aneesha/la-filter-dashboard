General Instructions

- This project is called LA-FILTER-DASHBOARD. It starts with a new repo. 
- LA-FILTER-DASHBOARD is a js and Python dashboard project. You can use Typescript and React
- You have full control over the github repo structure, the node modules you install and how you minify for a distribution. You'll need to update the .gitignore file. If you ever need to use .env files make sure that these can't be committed for privacy reasons. Just don't overwrite the prompts folder.
- When you plan, make sure you do functional decomposition, use appropriate design patterns and write tests. 
- Make sure the Readme.md file is up to date with how to install and use the lib 
- You are free to install libs that the project can use.
